# Resolve-App
Resolve app is an Onliner Complaint app. Here we can raise complaints regarding anything from online by going into this app
Online Complaint Registration and Management System

The Online Complaint Registration and Management System is a full-stack web application developed to simplify the complaint handling process for organizations. It enables users to register complaints, monitor their progress in real-time, communicate with assigned agents, and receive timely updates. Administrators can efficiently manage users, assign complaints to agents, monitor complaint resolution, and ensure smooth workflow across the platform.

Built using the MERN Stack (MongoDB, Express.js, React.js, Node.js), the project provides a secure, scalable, and user-friendly solution for digital complaint management with role-based access control for Users, Agents, and Admins.

 Key Features
 Secure User Authentication
 Complaint Registration
 Real-Time Complaint Status Tracking
 User-Agent Chat System
 Admin Dashboard
 Agent Dashboard
 Complaint Assignment & Management
 MongoDB Database Integration
 Responsive User Interface
 RESTful API Architecture
 Tech Stack
Frontend: React.js, Bootstrap, Material UI, Axios
Backend: Node.js, Express.js
Database: MongoDB, Mongoose
Authentication: JWT, bcrypt
Version Control: Git & GitHub
